# 锦江瞿溪路办公大楼会议签到APP

Use [`expo-router`](https://expo.github.io/router) to build native navigation using files in the `app/` directory.

## 🚀 运行

```sh
npx expo start
```

## ! 如果网络条件不佳，可以使用离线模式运行

```sh
npx expo start --offline
```
